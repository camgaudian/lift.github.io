# Front Squat

## Setup
- Set the bar on the front delts in a clean grip or cross-arm rack position.
- Elbows high; upper arms parallel to the floor if mobility allows.
- Feet shoulder-width; toes slightly out; core braced.

## Execution
- Squat down keeping the torso as upright as possible; elbows stay up.
- Knees track over toes; depth as mobility allows with a neutral spine.
- Drive through mid-foot to stand; maintain elbow height throughout.
- Breathe and brace at the top before the next rep.

## Common mistakes
- Elbows dropping — bar rolls forward; reduce weight.
- Excessive forward lean — often from ankle or thoracic mobility limits.
- Heels rising — elevate heels temporarily or work on ankle mobility.
- Losing rack position — adjust grip width or use straps for a clean grip.

## Safety notes
- Start with an empty bar to find a stable rack position.
- Stop for sharp wrist, shoulder, knee, or back pain.
- Consult a qualified coach for rack and mobility issues.
