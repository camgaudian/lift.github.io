# Face Pull

## Setup
- Attach a rope to a cable set at upper-chest to face height.
- Grip the rope with thumbs toward you; step back to create tension with arms extended.
- Stand in an athletic stance; brace the core and set shoulders down and back.
- Start with a slight lean back only if needed for balance, not to heave the weight.

## Execution
- Pull the rope toward the face by driving the elbows high and out to the sides.
- Externally rotate at the end so knuckles point back and hands finish beside the ears.
- Squeeze the rear delts and upper back; pause briefly without shrugging the neck.
- Return with control to full extension, maintaining cable tension.

## Common mistakes
- Pulling with the lower back by over-leaning — reduces rear-delt and rotator cuff work.
- Elbows staying low — turns the movement into a row instead of a face pull.
- Using too much weight — leads to jerking and poor shoulder mechanics.
- Shrugging the traps to finish the rep — shifts emphasis away from the target muscles.

## Safety notes
- Use light to moderate loads; this is a precision shoulder-health movement, not a max-effort pull.
- Stop for sharp front or side shoulder pain.
- Consult a qualified coach or clinician if you are rehabbing rotator cuff or impingement issues.
