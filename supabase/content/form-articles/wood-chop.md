# Wood Chop

## Setup
- Set a cable pulley at high position for a high-to-low chop, or low for low-to-high.
- Stand sideways to the stack with feet shoulder-width; knees slightly bent.
- Grip the handle with both hands; arms extended but not locked.
- Brace the core; hips and shoulders square before rotating.

## Execution
- Rotate through the torso and hips to pull the handle diagonally across the body.
- Keep arms relatively straight; power comes from the core and hips, not the arms.
- Control the return to the starting position against the cable tension.
- Exhale on the chop; maintain a stable lower body with minimal foot lift.

## Common mistakes
- Pulling with the arms only — reduces oblique and core engagement.
- Rotating only the arms without hip and thoracic involvement — limits power and range.
- Using excessive weight and losing control — increases lower-back stress.
- Allowing the feet to pivot excessively — reduces stability and control.

## Safety notes
- Start with light weight to learn the rotation pattern before progressing.
- Stop if you feel sharp lower-back, hip, or shoulder pain.
- Consult a qualified coach if you have disc issues or recent spinal injury.
